# Solana Vanity Wallet Generator

Generate custom Solana wallet addresses with your own prefix or suffix.

## How to Use

1. Click **Run** button
2. Open the **Webview** tab
3. Enter your desired prefix/suffix
4. Click **Start Generating**
5. Save your keypair when found!

## Features

- ✅ Real Solana Ed25519 keypairs
- ✅ Works with Phantom, Solflare, etc.
- ✅ Download keypair as JSON
- ✅ Case-sensitive option

## Tips

- Keep patterns to 2-4 characters for faster results
- Case-insensitive is ~1.7x faster
- Avoid: 0, O, I, l (not in Base58)

## Import Your Wallet

**Phantom:** Settings → Security → Import Private Key

**Solflare:** Add Wallet → Import → Private Key

⚠️ Never share your private key!
